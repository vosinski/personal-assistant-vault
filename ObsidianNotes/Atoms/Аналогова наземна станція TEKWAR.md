**Аналогова наземна станція від TEKWAR. Кейс 19**

- Дисплей 19ʼ без функції no blue no black: зображення зникає лише після повної втрати сигналу, літати можна і без окулярів, шуми та завади при наявності сигналу відтворюються «нормально».
- Модуль відеозахоплення — при підключенні телефону або ПК доступні стрімінг і запис.
- Два АКБ 8200 6s2p з можливістю роботи від двох АКБ (передбачена гаряча заміна) або від одного АКБ.
- Вхід HDMI.
- Порт заряджання для сторонніх девайсів 5V
- Вхід/вихід «тюльпан», вхід/вихід джек (кабель у комплекті), порт живлення для окулярів.
- Модуль-пустишка в пульт і кабель для з’єднання з кейсом.
- Пластиковий удароміцний кейс.
- MiniDVR.

**Виносна система аналогової наземної станції:**

-  **Патч (робочий діапазон 1080–1360) з пігтейлом**, відеоприймач 1.2 readytosky.Кастомний патч tekwar 4.9–6.0 19.5 dBi, кастомний хелікс 5.3–5.9 14 dBi, відеоприймач fusion.
-  **Порти для 2 ТХ на керування**, два порти додаткового живлення для ТХ керування, кабель додаткового живлення XT-30 → XT-30, антена ягі з центральною частотою 768 (робочий діапазон 700–850) або інша ягі за запитом, кріплення для другої ягі.
- **Фарбований алюмінієвий корпус із розміщенням усіх антен**, система фіксаторів для складання антен під час транспортування.
- **Кабель від кейсу до виносної системи довжиною від 30 метрів.**
- **Транспортувальний тубус.**
